## Application Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                    FINANCEBUDDY UI FLOW                        │
└─────────────────────────────────────────────────────────────────┘

Landing Page
    ↓
[Login/Register] → Authentication
    ↓
Dashboard (Main Hub)
    ↓
┌─────────────────────────────────────────────────────────────────┐
│                    MAIN NAVIGATION                             │
├─────────────────────────────────────────────────────────────────┤
│  Dashboard → Quick Calculator + Recent Predictions            │

```
